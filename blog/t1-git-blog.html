This is a brief overview of the benefits of version control

Version Control

There are several benefits to version control primarily in regard to collaboration between people on a software project. Version control allows the tracking of contributions by various team members towards a project. By tracking these contributions you can establish accountability and process around ensuring quality of your codebase.

Git

Git as a version control tool allows you to track changes to a codebase. Git allows you to take a copy of a codebase and modify it, taking snapshots of your progress as many times as you feel necessary.

GitHub

GitHub is a cloud based collaboration tool that allows you to store remote git repositories.
